🎮 Neon Rock Paper Scissors Arena

⚡ A modern, interactive Rock-Paper-Scissors game with neon UI, animations, AI opponent, and real-time game tracking.

🚀 Live Preview

👉 Open index.html to start the game
(Automatically redirects to the main game page)

✨ Features

⚡ Neon UI Design with smooth animations
🎯 Multiple Difficulty Levels (Easy, Medium, Hard)
🤖 AI Opponent with adaptive behavior
📊 Leaderboard System (Local Storage)
📜 Game History Tracking
🔥 Win Streak System
🎨 Dark / Light Theme Toggle

⌨️ Keyboard Controls
R → Rock
P → Paper
S → Scissors
T → Toggle Theme

💥 Advanced Visual Effects & Animations
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend (Optional): Node.js
Storage: Browser Local Storage

📁 Project Structure
📦 neon-rps-arena
 ┣ 📜 index.html        → Redirect entry
 ┣ 📜 rock.html         → Main game UI
 ┣ 📜 scissors.css      → Styling & animations
 ┣ 📜 paper.js          → Game logic & UI control
 ┣ 📜 server.js         → Node.js server
 ┣ 📜 package.json      → Project config
 
⚙️ Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/your-username/neon-rps-arena.git
cd neon-rps-arena
2️⃣ Run Locally (Simple)

Open:

index.html
3️⃣ Run with Node.js (Recommended)
npm install
npm start

🌐 Then open:
http://127.0.0.1:3000

🧠 Game Logic

Player selects: 🪨 Rock | 📄 Paper | ✂️ Scissors
AI generates move based on difficulty
Outcome rules:
Rock > Scissors
Paper > Rock
Scissors > Paper
Scores, streaks, and history update in real-time
